# Job Market Analysis & Opportunity Prediction
Data analysis project on 100K+ job listings to identify hiring trends, skill demand, salary patterns, and job market opportunities using Python and data visualization.

##  Project Overview
This project analyzes 1L+ job listings dataset to identify hiring trends,
skills demand, salary patterns, and job opportunities.

##  Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Features
- Data Cleaning
- EDA Analysis
- Skill Demand Analysis
- Salary Insights
- Visualization Dashboard

##  Dataset Size
100,000+ records
